# 应用简介

## 情景介绍

> 升级你的人生

简单一句话介绍这个应用的特色，就是*“游戏化的待办事项应用”*。



**用一个情景详细地介绍游戏化：**

> 完成了一篇文章，打开应用，在“写一篇文章”的事项前打上勾，标记完成。
>
> 此时你获得了一些「学识」和「创造」200经验值，并且「学识」升了一级。
>
> 应用还弹出了“成就解锁！法师”的成就解锁提示框。
>
> 此外，还**获得了10个金币**。
>
> 点击商店，查看为自己设置的奖励商品列表。
>
> 斟酌半刻，你购买了耗费8金币的奶茶。然后前往仓库，使用了刚刚购买的奶茶。
>
> 于是现实中，可以心满意足地点开外卖应用下单奶茶奖励自己。

该情景**主要只涵盖了应用的游戏化部分**，除此之外还有社区、统计、日历、桌面小部件等功能，会在后面部分介绍。

## 常见问题

**Q：**团队事项消失？

**A：**在退出登录的时候会清空未完成的团队事项。

可以查看自己加入的团队，**然后点击“领取”按钮重新领取团队事项。**

计划加入“一键领取所有已加入团队事项”的按钮。



**Q：**如何建立工作日事项或者周指定日事项?

**A：**重复频次选择“每日”后，选择“设置忽略”按钮。详情可以查看[建立事项](#建立事项)。



**Q：**没能收到事项的提醒？

**A：**默认的提醒用的是Android原生的闹钟机制，在国产OS里，这个闹钟机制可能需要应用后台运行才能正常运作。如果有精准的需求，**可以前往设置里，选择`更换提醒方式`，系统日历提醒会更准确，并且不受后台运行限制。**



## 绿色应用

并且，《人升》**无开屏、无内置广告、暂无内购**。体积也一直**尽量保持在6mb**左右。

并且通过了[Android 绿色应用公约 3.0验证](https://green-android.org/)，**无需读取手机状态、甚至无需外部存储空间权限**。

理论上运行**无需任何权限**，只为了一些可选功能申请了一些权限：

- **查看网络连接、查看WLAN连接、拥有完全的网络访问权限**

  为了实现登录、社区等功能。

  如果完全把《人升》当作一个离线工具使用，甚至可以关闭这些权限。

- **运行前台服务**

  为了实现桌面小部件完成事项的功能。

  同理，如果不需要用到桌面小部件，该权限也可以关闭。

- **日历读写权限**

  只有将提醒方式改为“系统日历提醒”的时候才会动态申请该权限。

  如果不需要“系统日历提醒”功能，该权限也可以关闭。
  
- **其他权限**

  一些其他权限诸如Play Store相关的，其实是第三方SDK申请的。

  比如Google授权登录、QQ授权登录。

## 建立事项

人升的事项依靠着**开始时间、期限时间和重复频次**的配合，可以建立出各种复杂情况的待办事项。

**以下是一些情况的例子：**

### 普通待办事项/琐事/购物清单

 {% asset_img task_01.jpg %}

这种类型的事项，不需要关心“期限时间”，重复频次默认也是“单次”。

所以，**不需要额外设置。只需要填好“待办事项”内容即可。**

### 习惯养成（每日早起、每周跑步等）

 {% asset_img task_02.jpg %}

调整**“重复频次”**为相应的“每日”或者“每周”后，再根据情况看要不要调整**“期限日期”**。

当“重复频次”不为“单次”的情况下，期限日期会默认为“当天”。

对于“每日”事项来说，这样设置一般是合理的。

而对于重复频次大于“每日”的事项，要根据具体情况调整期限：

> 1. 比如1号的时候建立“每周”事项，不改期限日期。那么下一次的事项的时间会在2-8号。
> 2. 如果你调整期限日期到2号。那么下一次的事项时间就会在3-9号。（相当于延后一天）

### 每周一-每周日

 {% asset_img task_03.jpg %}

通过期限时间和重复频次的配合，**你能轻松地设置出“星期一”到“星期日”的每周事项。**

**只要设置重复频次为“每周”，然后期限时间设置为“周末”即可~。**

人升弱化了开始时间的概念，在绝大多的时候可以让相应的开始时间自动计算得出。

### 当天23:00-第二天2:00

 {% asset_img task_04.jpg %}

通过开始时间、期限时间和重复频次的配合，你能轻松地设置出情况很复杂的待办事项。

只需要设置**开始时间为具体的“当天23：00”**，期限时间为**具体的“第二天2：00”**，**重复频次为“每日”**即可。

### 指定每周几（仅工作日、仅周末等）

 {% asset_img task_05.jpg %}

 {% asset_img task_06.jpg %}

设置“重复频次”为“每日”后，在“重复频次”的输入框右侧点击“设置忽略”。然后就能指定每周几重复了。

## 商店系统

{% note primary %} 

**上架商品，获得金币，购买奖励**

商店系统是对完成事项后奖励自我的一种游戏化体现，<del>并不是什么内购</del>。

 {% endnote %}

所以，在商店上架的商品，就是你打算在自己达到一定目标后给自己的奖励。

价格是你自己衡量是否达到这个目标的标尺。

<br />

 {% asset_img shop_01.jpg %}

### 金币获取手段

设置好了商品（奖励），那么就只差金币的获取手段了：

1. **自建待办事项，设置金币奖励**

   **自设奖励：**如果你对自己的金币体系很看重，那么最好每个事项都单独设置金币奖励。

   **自动计算：**否则的话，你可以选择`自动计算`金币奖励。

   `自动计算`会根据你设置的`重要程度`和`困难程度`计算出该事项的奖励，取值为15-120。

   然后你应该根据该取值定义商店商品的价格。

   当然，**你可以两种方式一起使用**。

2. **加入团队，完成团队任务获取奖励**

   如果团队的建造者设置了金币奖励，那么完成该事项的时候，你也能获得相应的金币。

3. **解锁成就**

   部分成就解锁后，会奖励小额金币。

<br />

### 仓库

默认情况下，购买的商品会放到仓库里。

需要点击`使用`的时候才消耗掉。

当然，最新版本，你能在购买的时候选择`立即使用`了。

<br />

### 贷款

右上角的商店设置里，可以设置贷款金额。

现在的贷款实现，其实是允许金币负值。

比如，你设置了贷款1000。那么就是允许金币最小为-1000。



所以，偿还贷款的方式，就是挣金币~

<br />

### 使用参考

具体上架什么商品，取决你自己的爱好~

**这里提供一些参考：**

奶茶 —— 10金币：使用之后，奖励自己一杯奶茶

空气 —— 0金币（免费）：要多少有多少

手机 —— 3000金币：换台新手机

**另外，也有一些用户把商品当作惩罚来用😂：**

浪费时间惩罚 —— 5金币：如果今天浪费了时间，就购买这项金币惩罚

## 社区

{% note primary %} 

**组织团队、共同进步**

《人升》的社区也不是普通的论坛，而是一个组织团队共同完成某个事项，养成习惯的地方。

并且这里主打轻社交，没有聊天，没有通知。

但是你能看到浏览别人完成事项的动态。

 {% endnote %}

### 团队

社区的主体就是团队，

你可以浏览加入感兴趣的团队。

如果找不到，那么可以自己建立一个团队。

>  团队的内容是公开的，**所以请不要建立比较私人的事项，或者涉及违法敏感相关的事项，谢谢~**。
>
>  如果有特殊需要，可以建立成本地事项。

<br />

加入或者建立一个团队之后，会自动领取团队事项。

**如果你删除错了团队事项，或者退出登录导致团队事项消失。**

那么，可以返回团队详情界面，**点击`领取`按钮**重新领取团队事项。

<br />

团队事项暂时不支持编辑、放弃等操作。

后续会开放**部分编辑**功能。

<br />

### 新建团队

新建团队的界面其实和新建待办事项差不多，只是缺少了一些自定义选项。

值得注意的是，

“`首次开始日期`+`首次开始时间`”和“`首次结束日期`+`结束时间`”指的是**第一次事项的开始结束时间，并不是团队的结束时间**。

比如，每日事项，这两个时间一般来说是一天的00:00:00到23:59:59即可。

如果是`早起`之类的事项，设置为**同一天**的06:00:00到08:59:59。

再比如说每周事项，可以设置为周一00:00:00到周日23:59:59。

<br />

**如果需要终止团队，**

如果创建者到团队详情页面右上角选项，选择`终止`即可。

如果你不是创建者，可以选择`退出`团队。

<br />

### 时刻

在完成团队事项的时候，你可选择性地发表动态，并且可以贴上图片。

> 为了节省服务器流量，以及防止一些问题，图片会默认进行压缩。

你能在时刻页面浏览到其他人发表的动态内容，查看别人坚持事项的情况。

同时，你也能关注感兴趣的人。并且筛选关注的人的动态等。



在个人主页（**入口**：`我的`→`我的主页`→`查看`），

你能随时浏览自己发表过的动态，长按也可以进行删除操作。

<br />

### 排行榜

你可能会诧异为什么排行榜只有你自己一个人。😂

因为并不想让排行榜变成一个攀比的地方，所以设计成了只显示自己与自己关注的人。

<br />

**所以，去关注一下其他人吧~<del>（说不定还能解锁成就）</del>**



排行榜目前功能较薄弱，以后可能会改进。
